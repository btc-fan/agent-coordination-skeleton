# Validation Checklist

A task is not done unless:
- task-specific DoD is satisfied
- requirements artifact exists and matches shipped scope
- relevant ADRs are updated for major decisions
- verification evidence exists (tests/lint/typecheck/repro steps as applicable)
- no unresolved reviewer rejection remains
- required docs are updated
- issue/PR linkage is complete
- no pending clarification blocks the task
- memory updates were written where required
- routing/ops noise was kept out of active coding context
